ECG Portal - Professional UI + Backend (Node.js + SQLite)
-------------------------------------------------------
How to run:
1. Install Node.js (LTS)
2. Open terminal, go to backend folder:
   cd ecg-portal-final/backend
3. Install deps:
   npm install
4. Start server:
   npm start
5. Open in browser: http://localhost:5000
